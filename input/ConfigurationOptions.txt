<VerbosityLevel>

 The VerbosityLevel node specifies the degree of information which is printed out on the screen.
 The supported verbosity levels are listed in the table below:
 
     Level         Description 
 
     ALWAYS        Default verbosity level
     VERBOSE
     DEBUG
     INFO
 
<ExecutionMode>

 The ExecutionMode node specifies the main execution mode of Fittino. If OPTIMIZATION is set,
 Fittino tries to find the optimal parameter set of a given model w.r.t. experimental observation.

<Model>

 The Model node specifies the model to be optimized. The supported models and their parameters are
 listed in the table below:
 
     ModelType     Parameters
 
     HEC
     CMSSM         A0, M0, M12, TanBeta
     Rosenbrock    Dimensions, X1 - X4
     
<Optimizer>
 
 The Optimizer node specifies the optimization algorithm and several general optimizer settings.
 Supported optimizers are:

     OptimizerType
 
     GeneticAlgorithm
     Minuit
     ParticleSwarm
     SimulatedAnnealing

<Sampler>

 The Sampler node specifies the sampling algorithm and several general sampler settings. Supported
 samplers are:

     SamplerType
 
     MarkovChain 
     Simple
